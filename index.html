הגדרה ראשונית (ללא עריכת קבצים)

  1. ב־Firebase Console הוסיפי Web App וקבלי את firebaseConfig.
  2. הדביקי אותו כאן כ־JSON (האובייקט שמתחיל ב-{ apiKey: ... }).
  3. הוסיפי אימיילי אדמין (לאישור אירועים).
  4. שמרי והעמוד ייטען מחדש.

ההגדרות נשמרות ב-LocalStorage בדפדפן.
